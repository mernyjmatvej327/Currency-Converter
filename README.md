# Currency Converter

**Автор:** [Мерный Матвей Александрович]

## Описание

Графическое приложение для конвертации валют с использованием фиксированных курсов. Программа позволяет:
* выбирать валюты для конвертации («из» и «в»);
* вводить сумму для конвертации;
* получать результат расчёта;
* просматривать историю последних 10 конвертаций;
* очищать историю.

## Особенности

* **Графический интерфейс** на базе `tkinter`.
* **Фиксированные курсы валют** (можно обновлять вручную в коде).
* **Сохранение истории** конвертаций в JSON‑файл.
* **Проверка корректности ввода:** программа проверяет, что сумма — положительное число.
* **Обработка ошибок:** пользователь получает понятные сообщения при некорректном вводе.

## Установка и запуск

### Требования

* Python 3.6 или выше.
* Стандартные библиотеки Python (устанавливать не нужно):
  * `tkinter` — для графического интерфейса;
  * `json` — для работы с историей конвертаций;
  * `os` — для проверки существования файла истории;
  * `datetime` — для отметки времени конвертации.

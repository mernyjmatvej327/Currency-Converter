# Currency Converter (Конвертер валют)

**Автор:** [Мерный Матвей Александрович]

## Описание

Графическое приложение для конвертации валют с использованием актуальных курсов через API, сохранением истории операций в JSON‑файл и возможностью её очистки.

## Функционал


* **Графический интерфейс** на базе `tkinter` (входит в стандартную поставку Python).
* **Актуальные курсы валют** через API `exchangerate-api.com`.
* **Выбор валют:** из и в (USD, EUR, GBP, JPY, CAD, AUD, CHF, CNY, RUB).
* **Поле ввода суммы** с проверкой корректности.
* **Кнопка конвертации** для расчёта результата.
* **Отображение результата** в интерфейсе.
* **История последних 10 операций** в виде таблицы.
* **Сохранение истории** в файл `conversion_history.json`.
* **Очистка истории** кнопкой в интерфейсе.

## Как получить API‑ключ

1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Нажмите «Get Free API Key» (получить бесплатный API‑ключ).
3. Введите email — ключ придёт на почту мгновенно.
4. Скопируйте полученный ключ.
5. Вставьте ключ в код вместо `YOUR_API_KEY` в строке:

   self.api_key = "YOUR_API_KEY"
